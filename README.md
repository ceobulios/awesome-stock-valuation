# Awesome Stock Valuation

A curated list of tools and resources for stock valuation and fair value analysis.

## Fair Value Tools

- **[Fair Price Index](https://fairpriceindex.com)** — Fair value for 37,000+ stocks using DCF, relative valuation, and analyst consensus. Free, updated daily.
- [Alpha Spread](https://alphaspread.com) — Intrinsic value calculator with machine learning
- [Simply Wall St](https://simplywall.st) — Visual stock analysis and valuation
- [Finviz](https://finviz.com) — Stock screener and visualization
- [TIKR](https://tikr.com) — Financial data terminal for investors
- [Stock Unlock](https://stockunlock.com) — Free DCF calculator
- [Morningstar](https://morningstar.com) — Analyst-driven fair value estimates

## Educational Resources

- [What is Intrinsic Value?](https://www.investopedia.com/terms/i/intrinsicvalue.asp) — Investopedia explainer
- [DCF Analysis Guide](https://www.investopedia.com/terms/d/dcf.asp) — How discounted cash flow works
